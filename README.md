## :computer: Projeto

The application consists of a SPA with the Universe theme.

Concepts were used, such as:
- SPA concepts;
- Route mapping;
- Asynchronous and promises;
- Object orientation;
- Classes;
- Creation of a simple local server with NodeJs.
